お祭り人数カウントアプリ

開発背景
出身小学校の祭りを運営する中で、「来場者数をリアルタイムで把握できない」という課題がありました。
1回目の開催では想定1,000人に対し2,400人が来場し、混雑が発生。
2回目の改善に向けて、属性別(大人・子ども・高齢者)の来場者数を
リアルタイムで記録・分析できるツールを開発しました。

主な機能
属性別カウント 大人・子ども・高齢者を個別にカウント
リアルタイム集計 合計人数を自動計算
時間帯別推移グラフ　Chart.jsで可視化
CSV出力　後から詳細分析が可能
データ永続化 localStorageで記録を保持

技術スタック
HTML/CSS/JavaScript
Chart.js (グラフ描画)
localStorage (データ永続化)

実際の活用
2回目の祭り運営で実際に使用し、時間帯別の来場者分析に活用。
混雑ピーク時間を把握することで、効果的な人員配置を実現しました。
